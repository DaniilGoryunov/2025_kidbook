1. Владелец (Owner)
 • Центральный субъект системы
 • Основные функции: кормление, уход, игры, посещение ветеринара
2. Домашнее животное (Pet)
 • Основной объект заботы
 • Классифицируется по видам
3. Ветеринар (Veterinarian)
 • Медицинский специалист
 • Оказывает профессиональную помощь
4. Зоомагазин (Pet Store)
 • Торговая точка снабжения
 • Обеспечивает необходимыми товарами
5. Корм (Pet Food)
 • Базовый ресурс содержания
 • Различается по типам питания
6. Собака (Dog)
 • Один из видов питомцев
 • Требует специального ухода
7. Кошка (Cat)
 • Популярный вид домашних животных
 • Имеет специфические потребности
8. Попугай (Parrot)
 • Представитель птичьих
 • Нуждается в особых условиях
9. Хомяк (Hamster)
 • Мелкий грызун
 • Требует специального оборудования
10. Ошейник (Collar)
 • Атрибут собаки
 • Используется для контроля
11. Лоток (Litter Box)
 • Гигиеническое приспособление
 • Необходим для кошек
12. Клетка (Cage)
 • Жилище для птиц
 • Обеспечивает безопасность
13. Колесо (Exercise Wheel)
 • Тренажёр для грызунов
 • Поддерживает активность
14. Выгул (Walking)
 • Процедура для собак
 • Обеспечивает физическую нагрузку
15. Осмотр (Check-up)
 • Медицинская процедура
 • Проводится ветеринаром
16. Прививки (Vaccinations)
 • Профилактические меры
 • Обеспечивают здоровье
17. Игрушки (Toys)
 • Средства развлечения
 • Поддерживают активность
18. Аксессуары (Accessories)
 • Дополнительные предметы
 • Упрощают уход
19. Сухой корм (Dry Food)
 • Один из видов питания
 • Удобен в использовании
20. Влажный корм (Wet Food)
 • Альтернативный тип корма
 • Близок к натуральному
21. Натуральный корм (Natural Food)
 • Естественный рацион
 • Требует подготовки
